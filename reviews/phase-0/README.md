# Phase 0 external reviews

This directory stores independent review evidence. The drafter and final accountant reviewer should be different people.

Required before Phase 1 authorization:

1. Accountant/CA red-team report.
2. Two founder workflow walkthrough notes.
3. Recorded objections and dispositions.
4. Final go/no-go entry appended to `docs/phase-0/DECISION-LOG.md`.
